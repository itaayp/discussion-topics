    This is the Topic Controller and will orchestrate the actions related to the Topic

    Functions

    create(conn, map)

    This function adds a new row in the topics table on the database

    delete(conn, map) 
    This function deletes a topic from the database
    edit(conn, map)

    This function is responsible to redirect the user to the Edit Topic screen. The argument %{"id" => topic_id} is the id of the topic selected by the user to edit. This argument is part of the params argument

    index(conn, params)

    This function displays all the topics records from the database

    new(conn, params)

    This function renderizes the New Topic screen. The changeset is needed to be passed by parameter when calling the render function and it means: "make availble this values, so I can use it on my page"

    update(conn, map) 
    This function updates the value on the database of a topic already created
